1. Executive Summary (Non-Technical)

Spectre is a critical security vulnerability affecting modern computer processors from Intel, AMD, and ARM. It exploits a feature called "speculative execution," where the processor tries to guess future commands to increase speed. Attackers can trick the processor into guessing wrong and momentarily accessing private data (like passwords). Although the processor discards the wrong guess, the data leaves faint traces in the system's memory cache, which attackers can measure and reconstruct.
